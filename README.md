# Ascii Webcam

Display ascii art (ish) frames from the webcam in the terminal.

## Prerequisites

- numpy
- opencv-python
- scipy

## Use

Simply run python cam.py.

You might want to resize the text in your terminal to get it to work.
I get about 30 FPS on my laptop.
